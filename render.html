Render 操作說明

什麼是 Render？

Render 是一個可以免費部署靜態網站的雲端平台。我們可以把放在 GitHub 上的網頁檔案，透過 Render 自動變成一個對外公開的網站！

操作步驟（靜態網站）

  1. 前往 https://render.com，註冊帳號（可以使用 GitHub 登入）
  2. 登入後點選「New → Static Site」
  3. 選擇要部署的 GitHub Repository（建議事先設為 public）
  4. 填寫資料：
    • Name： 你的網站名稱
    • Branch： main 或 master
    • Build Command：（空白即可）
    • Publish Directory： 輸入 ./
  5. 點擊「Create Static Site」開始部署
  6. 完成後會得到一個網址，例如：https://你的網站名稱.onrender.com

常見錯誤排除

  • Render 無法找到檔案？請確認你的 GitHub repo 裡有 index.html 並放在最外層
  • 網址打開是空白？請檢查 HTML 是否有語法錯誤，或重新部署一次
