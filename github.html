GitHub 操作說明

一、建立 GitHub 帳號

  1. 前往 https://github.com
  2. 點選右上角「Sign up」，依指示註冊帳號
  3. 驗證電子信箱後即可開始使用

二、建立新儲存庫（Repository）

  1. 登入後點右上角「+」 → 「New repository」
  2. 輸入儲存庫名稱（例如：ai-camp-site）
  3. 選擇「Public」，勾選「Add a README file」
  4. 點擊「Create repository」

三、上傳網頁檔案

  1. 點進儲存庫頁面 → 點選「Add file」 → 「Upload files」
  2. 選擇 `index.html`、`intro.html` 等檔案上傳
  3. 下方輸入簡單說明，例如 "上傳首頁" → 點擊「Commit changes」

四、啟用 GitHub Pages

  1. 在儲存庫頁面 → 點選上方「Settings」
  2. 左邊選單點「Pages」
  3. 在「Branch」選擇 main，資料夾選「/ (root)」
  4. 按下「Save」，幾秒後會產生一個網址
  5. 你的網站就會在這個網址上線！

範例網址

https://你的帳號.github.io/你的儲存庫名稱/
（例如：https://aicamp123.github.io/ai-camp-site/）
