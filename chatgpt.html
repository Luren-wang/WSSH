ChatGPT 操作說明

一、如何開始使用

  1. 前往 https://chat.openai.com
  2. 使用 Google 帳號或 Apple ID 註冊 / 登入
  3. 進入後即可輸入問題開始聊天

二、適合問 ChatGPT 什麼？

  • 寫程式、解錯誤、解釋程式碼
  • 撰寫報告開頭、修改句子
  • 查資料、問冷知識
  • 練習英文對話、寫履歷、自我介紹

三、提問技巧

與 ChatGPT 對話時，越清楚具體，回覆會越好：

不佳：
「幫我寫一個網頁」
較佳：
「請幫我寫一個包含3個按鈕、背景為藍色的 HTML 網頁」
不佳：
「怎麼用 GitHub？」
較佳：
「請一步一步教我：怎麼用 GitHub 上傳 index.html 並公開成網站」

四、ChatGPT 和 Google 有什麼不同？

  • Google： 搜尋資料、新聞、找網站
  • ChatGPT： 幫你整理、寫作、程式設計、解釋觀念

五、常見用途範例

「幫我解釋 HTML 中 <a> 標籤的用法」
「請幫我改寫這段自我介紹，語氣更自然」
「我想做一個介紹旅遊景點的網頁，幫我設計內容和排版」
